# αἱρέω

προσδιαιρέω = προσ + δι+ αι+ ρέω

συγκαθαιρέω = συγ + καθ + αιρέω

προσκαταισχύνω = προσ-κατ-αι-σχύνω

προαναβαίνω = προ-αν-α-βαίνω
προσαναβαίνω = προσ-αν-α-βαίνω

προαναλίσκω = πρ-ο-αναλίσκω
προεξαναλίσκω = προ-εξ-αναλίσκω
προκαταναλίσκω = προ-κατ-αναλίσκω

προσαναμιμνήσκω - προσ-αναμιμνήσκω - προσ-ανα-μιμνήσκω

προσανίσταμαι = προσ-ανίσταμαι

παραναβαίνω = παρ-ανα-βαίνω

συμπαραναλίσκω = συμ-παρ-αναλίσκω

παραβάλλω = παρ-α-βάλλω

ἀντιπαραγράφω = ἀντι-παρα-γράφω

παρεπιγράφω = παρ-επι-γράφω

προσκαταγράφω = προσ-κατα-γράφω

ὑποκατάστατος = ὑπο-κατά-στατος

παρυπολαμβάνω = παρ-υπο-λαμβάνω

προσυπογράφω = προσ-υπο-γράφω

παρυπομιμνήσκω = παρ-υπο-μιμνήσκω

ἐπεξήγησις = ἐπ-εξ-ήγησις

ἐξελθεῖν = ἐξ-ε-λθεῖν
